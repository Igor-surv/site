Git

O que é GIT?

O git é um sistema de controle de versões distribuído que permite aos desenvolvedores rastrear e gerenciar alterações em seu código-fonte ao longo do tempo

Por que usar o GIT?

O git permite que você rastreie todas as alterações feitas no código

O git serve como um backup de seu código e sua flexibilidade permite trabalhar em diversas ramificações

Ele facilita a colaboração entre desenvolvedores, permitindo que eles trabalhem em diferentes ramificaçõe do código

Historia do GIT

O Git foi criado por Linus Torvalds em 2005.

Ele foi desenvolvido inicialmente para gerenciar o código-fonte do kernel do Linux.

O Git foi criado em Helsinki, Finlândia, e foi inicialmente hospedado no servidor de desenvolvimento do kernel do Linux.

Comandos básicos do Git

  • git init: Inicializa um novo repositório Git.
  • git add <arquivo>: Adiciona um arquivo ao staging.
  • git commit -m "mensagem": Cria um novo commit com uma mensagem.
  • git log: Exibe o histórico de commits.
